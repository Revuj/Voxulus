Voxulus

Voxulus is browser extension that helps people with motor disabilities have an easier time navigating the web, using not only head movement but also voice commands.

Voice Commands

  • Search the Internet
    • start search/new tab - After requesting the search parameters it'll perform a Google search of them
  • Click Something
    • click - After requesting the commands the element directly below the head movement pointer is left clicked
  • Scroll a Page
    • scroll down - Move down the content of a website
    • scroll up - Move up the content of a website
    • stop - Stop moving the content of a website
    • faster - Increase the speed the content of a website is moving
    • slower - Decrease the speed the content of a website is moving
  • Zoom
    • zoom out - Zooms out the current tab
    • zoom in - Zooms in the current tab
  • Browser Control
    • go back - Goes back in the browser history
    • go forward - Goes forward in the browser history
    • reload - Reloads the current page
  • Write Text
    • start writing - Starts listening for speech to transform to text
    • stop writing - Stops listening for speech
    • erase/delete/undo - Removes the last added input
    • start spelling - Starts listening for speech to transform to characters (you can use words starting with the character you want to input)
    • stop spelling - Stops listening for speech
    • next - Selects the next input element
    • submit - Submits the current form
  • Play Media Content
    • play - Plays media content
    • pause - Pauses media content
    • back [time] seconds - Rewinds the media content [time] seconds
    • forward [time] seconds/skip [time] seconds - Forwards the media content [time] seconds
    • volume up - Increases volume level
    • volume down - Decreases volume level
  • Tab Management
    • close tab - Closes the current tab
    • close tab [number] - Closes the tab at position [number]
    • select tab [number] - Switches to the tab at position [number]
  • Special Characters
    • newline - Changes to a new line in the current input field
    • whitespace - Adds a whitespace ("") to the current input
    • at sign - Adds a at sign ("@") to the current input
    • dot - Adds a dot (".") to the current input
